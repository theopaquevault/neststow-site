隐私政策

最后更新日期：2026年XX月XX日

欢迎使用 Neststow（以下简称“本App”）。我们非常重视您的隐私。本隐私政策旨在向您清晰说明本App如何处理您的信息。

核心承诺：本App是一款优先保护隐私的工具。我们不需要您创建账户，也不收集、存储或访问您的个人物品数据。您的所有物品信息（名称、日期等）均仅存储在您的本地设备中。若您主动开启iCloud同步功能，相关数据将存储于您的个人iCloud账户中，我们无法获取或查看。

请您在使用本App前仔细阅读本政策。若您不同意本政策，请勿使用本App。

1. 信息的收集

1.1 我们不收集的信息

  • 个人身份信息（如姓名、邮箱、电话号码）。
  • 您在App中录入的物品数据、到期日等内容。
  • 精确的地理位置信息。
  • 健康、财务或敏感个人信息。

1.2 我们自动收集的信息（匿名化）

为了提升App性能、修复错误和优化用户体验，我们集成了 Firebase Analytics 工具。该工具会自动收集匿名的使用统计数据，包括：

  • 设备型号、操作系统版本。
  • App内浏览的页面、点击的按钮。
  • App崩溃报告。
  • 大致的地理区域（仅国家/地区级别）。

此类信息无法用于识别您的个人身份。

1.3 崩溃报告（Firebase Crashlytics）

为了提升App的稳定性并及时修复崩溃问题，我们集成了 Firebase Crashlytics 工具。该工具会在App发生崩溃时自动收集以下技术信息：

  • 崩溃日志和堆栈跟踪
  • 设备型号、操作系统版本
  • App版本号
  • 崩溃发生时间

此类信息无法用于识别您的个人身份，仅用于定位和修复App的技术问题。

1.4 远程配置（Firebase Remote Config）

我们使用 Firebase Remote Config 来远程管理App的部分配置参数，以便在不发布新版本的情况下优化功能体验。该服务会收集匿名的配置元数据，无法用于识别您的个人身份。

1.5 文字识别功能（Google ML Kit）

本App提供文字识别功能（如识别购物小票上的文字），此功能由 Google ML Kit 提供。当您使用该功能时，ML Kit 会收集以下技术数据用于诊断和服务优化：

  • 设备信息（如制造商、型号、操作系统版本）
  • 应用包名和版本
  • 性能指标（如识别延迟）
  • 错误代码和事件类型

此类信息无法用于识别您的个人身份。识别过程在设备本地或通过加密传输完成，ML Kit 不会将数据转移给第三方。

1.6 订阅管理（RevenueCat）

本App的订阅功能由 RevenueCat 提供技术支持。当您购买或管理订阅时，RevenueCat 会收集必要的购买信息，包括购买记录和订阅状态。此类信息仅用于验证您的订阅权限和提供相关服务功能。RevenueCat 不会收集您的支付信息（如信用卡号）或健康、位置等敏感数据。

2. 信息的存储

  • 本地存储：您的所有物品数据默认存储在您的手机本地。
  • iCloud 同步：您可在App设置中选择开启iCloud同步。开启后，您的数据将同步至您个人的iCloud账户中，由 苹果公司（Apple Inc.） 依据其iCloud条款进行存储和加密。我们无法获取、访问或控制您的iCloud数据。
  • 存储期限：您的数据将一直保留在您的本地设备或iCloud账户中，直至您主动在App内删除数据，或从设备中卸载本App。

3. 信息的共享与披露

我们不会向任何第三方出售、出租或交易您的个人信息。

仅在以下有限情形下可能涉及信息处理：

  • 服务提供商：本App集成了Firebase系列服务（Analytics、Crashlytics、Remote Config）、Google ML Kit及RevenueCat，这些服务仅收集必要的技术数据或购买信息，无法用于识别您的个人身份。
  • 法律要求：如法律强制要求，我们可能需要配合提供信息。鉴于我们通常不掌握您的个人数据，您可能需要自行从设备中提供相关信息。

4. 到期提醒功能特别说明

本App的提醒功能基于您设备的本地通知系统实现，提醒的触发受以下因素影响：

  • 系统限制：iOS系统对单个App的待处理通知数量有限制（最多64条）。若超出此数量，最早的提醒可能被系统自动取消。
  • 设备状态：省电模式、专注模式、通知权限关闭等可能导致提醒延迟或失效。

请注意：本App仅作为辅助管理工具。对于涉及健康、安全的物品（如药品、易腐食品），请勿仅依赖本App的提醒。建议您同时使用系统日历或闹钟进行交叉确认。因完全依赖本App提醒而造成的任何损失，我们不承担责任。

5. 您的权利

由于所有个人数据均存储在您的设备上，您可通过以下方式直接行使您的权利：

  • 访问/更正：在App内直接查看、编辑或删除您的物品条目。
  • 数据删除：卸载本App即可永久删除所有本地数据。iCloud中的数据需您自行登录iCloud账户管理。
  • 关闭统计：您可在手机系统设置中关闭App的数据追踪权限（如iOS的“允许App请求跟踪”）。

6. 隐私政策的更新

我们可能会更新本政策。若有重大变更，我们将在App内通过弹窗等方式通知您，并在必要时获取您的明确同意。

7. 关于第三方服务的说明

本App集成了以下第三方服务：

  • Firebase Analytics：使用统计分析
  • Firebase Crashlytics：崩溃报告
  • Firebase Remote Config：远程配置管理
  • Google ML Kit：设备端文字识别
  • RevenueCat：订阅管理

上述服务均由Google（Firebase/ML Kit）或RevenueCat提供，仅收集必要的技术数据或购买信息，无法用于识别您的个人身份。您可通过各服务提供方的官方隐私文档了解更多信息。
